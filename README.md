Set Up φορμας επικοινωνιας:



Χρησιμοποιώ το web3forms το οποιο είναι δωρεαν για απλη χρήση και πολύ ευκολο στο setup.



Link: https://web3forms.com/platforms/html-contact-form#step\_\_1



1) Κανοντας Create Form, αφου παρετε το προσωπικο σας id το αντικαθίσταται με το δικο μου και απευθειας κάθε φορα που συμπληρώνεται η φορμα επικοινωνιας τα μηνυματα ερχονται στο προσωπικο σας email.



2) Το id το βαζεται στην γραμμή #153 στις παρενθεσεις του value="id" (<input type="hidden" name="access\_key" value="0c2eb960-ecdb-456c-a1e7-89cae510b30c">) 



3) Χρειαζεται login για να παρετε προσωπικο id.
